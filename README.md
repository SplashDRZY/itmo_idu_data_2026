# Исследование потенциала создания и развития локаций технических видов спорта как инструмента легализации уличных гонок

![Картинка](race.jpg)

### Краткое описание
- Исследование направлено на анализ проблемы уличных гонок (street racing) и предложения методологии их легализации и профилактики в рамках российского законодательства.
- Основано на нормативной базе РФ, сравнительном правовом анализе иностранных практик, а также кейсах зарубежных стран.
- Включает обзор методов борьбы с опасным вождением, а также идеи по созданию локаций для безопасных соревнований.
